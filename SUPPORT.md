# Support

Aegora is currently in the foundation phase.

## Getting Help

Use GitHub issues for bug reports, reproduction cases, documentation gaps, feature proposals, and mentored contribution requests.

Before opening an issue, please search existing issues to avoid duplicates.

## Good Support Requests

Include:

- What you are trying to build
- What happened
- What you expected
- A minimal reproduction when possible
- Aegora, Bun, and TypeScript versions

## Community Expectations

Support should be respectful in both directions.

People asking questions should provide context when they can. People answering questions should assume good intent and avoid shaming newcomers.

## Commercial Support

Aegora does not currently offer official commercial support.
